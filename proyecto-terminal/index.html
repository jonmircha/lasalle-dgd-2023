Proyectos

Documento

Objetivo: Generar un documento que permita a cada alumno determinar la estructura de la investigación, el desarrollo conceptual y conformación de una propuesta gráfica, como estrategia de comunicación visual y gráfica, mediante diferentes procesos y métodos de diseño desde un enfoque para la aplicación de recursos y elementos que le permitan una propuesta integral.

Descripción: El documento del proyecto terminal I, según las características propias de cada caso, se deberán tomar en cuenta los siguientes elementos: Estructura formal: Formato carta. Estructura del documento: Portada con título, introducción, índice, los tres primeros capítulos, citación APA y fuentes de investigación.

Entregable: Trabajar sobre el documento en cada clase y respetar las fechas de entrega y asesorías correspondientes.
